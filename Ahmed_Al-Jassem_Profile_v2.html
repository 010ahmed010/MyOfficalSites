SQL Injection

High

Discovered SQL injection vulnerability in a web application's login form that allowed bypassing authentication.

Impact:

  • Unauthorized access to user accounts
  • Potential extraction of sensitive database information
  • Complete database compromise in some cases

Remediation:

  • Implemented parameterized queries

  • Deployed Web Application Firewall (WAF)

Medium

Impact:
